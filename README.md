## Golang

```
$ wget https://golang.org/dl/go1.14.6.linux-armv6l.tar.gz
$ sudo tar -C /usr/local -xzf go1.14.6.linux-armv6l.tar.gz

$ echo export 'PATH=$PATH:/usr/local/go/bin'  >> ~/.profile
$ source ~/.profile

$ go version
go version go1.14.6 linux/arm

$ cd golang/
$ go build main.go 
$ ./main
```
